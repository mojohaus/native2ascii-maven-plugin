Plugin Documentation

Goals available for this plugin:

Goal Description
native2ascii:help Display help information on native2ascii-maven-plugin.
Call mvn native2ascii:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
native2ascii:inplace Converts files with characters in any supported character encoding to one with ASCII and/or Unicode escapes.

This mojo converts files inplace, it is not recommended to execute it on src subdirectories.

native2ascii:resources Converts files with characters in any supported character encoding to one with ASCII and/or Unicode escapes.
native2ascii:testResources Converts files with characters in any supported character encoding to one with ASCII and/or Unicode escapes.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.2
JDK 1.8

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.codehaus.mojo</groupId>
          <artifactId>native2ascii-maven-plugin</artifactId>
          <version>2.1.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.codehaus.mojo</groupId>
        <artifactId>native2ascii-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
